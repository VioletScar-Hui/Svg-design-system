# Changelog

## v1.0.0 - 2026-06-09

Initial public release of `svg-design-system`.

### Added

- Codex / Claude-compatible `SKILL.md` for SVG diagram design.
- 10 named three-color palettes with semantic usage notes.
- Design token references for canvas, typography, spacing, strokes, shadows, and light/dark adaptation.
- Diagram recipes for flowcharts, matrices, architecture diagrams, timelines, networks, cycles, and comparison charts.
- Feishu/Lark whiteboard-safe SVG pipeline and constraints.
- Sample light and dark SVG cards.
- 10 SVG palette examples for the README gallery.
- Separate English and Chinese README files at the repository root and skill-directory level.
